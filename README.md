# 📈 Stock Price Prediction using Linear Algebra

## 📝 Project Overview

This project predicts the **next day stock price** of a company using historical stock market data using **Linear Regression**.  
Linear Regression is a Machine Learning algorithm based on **Linear Algebra** concepts.

The system studies previous stock trends and predicts the future price movement.

It also provides:

- Next Day Predicted Price  
- Percentage Increase / Decrease  
- BUY / HOLD / SELL Suggestion  
- Graphical Trend Analysis  

---

# 🎯 Project Objective

The main objective of this project is:

- To apply Linear Algebra in a real-world financial application  
- To predict stock prices using historical data  
- To understand Machine Learning concepts  
- To visualize stock market trends  
- To help users make basic investment decisions
